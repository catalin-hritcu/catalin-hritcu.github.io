Formally Secure Compartmentalizing Compilation

Course at 18th International School on Foundations of Security Analysis and Design (FOSAD) in Bertinoro, Italy on 27-28 August 2018

Lecturer: Cătălin Hriţcu

Helpers: Jérémy Thibault, Carmine Abate, Roberto Blanco, Florian Groult

Schedule and Slides

Monday, 27 August 2018:
  • 9:00--10:00 - Introduction to Formally Secure Compartmentalizing Compilation
  • 10:00--11:00 - Building a simple compilation chain
  • 15:00--16:30 - Correct Compilation, Linking, Components
  • 16:30--17:00 - Secure compilation for unsafe languages
Tuesday, 28 August 2018:
  • 11:30--13:30 - Secure interoperability with lower-level code
